## About ğŸ“

**Weather Now** is a modern, responsive web app that shows current weather, daily forecasts, and hourly forecasts worldwide.  
The app detects the user's city via IP for first-time visitors and allows dynamic searching for other cities.  

Built with **HTML, JavaScript, and Tailwind CSS**, it focuses on **clean UI, accessibility**, and real-time weather data via **Open-Meteo API**.
