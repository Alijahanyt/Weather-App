# Weather Now üå§Ô∏è

**Weather Now** is a modern, responsive web application that shows current weather conditions, daily forecasts, and hourly forecasts for any city worldwide. The app automatically detects the user's location based on their IP for first-time users and allows dynamic searching for other cities.  

The app is built with **HTML, JavaScript, and Tailwind CSS**, with a focus on clean UI, accessibility, and real-time weather data fetched from **Open-Meteo API**.

---

## Features

### üåü Core Features
- Displays **current weather**: temperature, feels like, humidity, wind, precipitation, and weather icons.
- Shows **daily forecast** for 7 days, including max/min temperatures.
- Displays **hourly forecast** with icons and temperatures for every hour.
- Automatically detects user's city via **IP-based location** for first-time visitors.
- **Dynamic search** for cities with live updates on weather.
- **Units conversion**:  
  - Temperature: ¬∞C / ¬∞F  
  - Wind speed: km/h / mph  
  - Precipitation: mm / in  

### üé® UI/UX Features
- Fully **responsive design** using Tailwind CSS.
- **Modern dark theme** with smooth gradients.
- **Custom styled radio buttons** for units selection.
- **Accessible**: works with screen readers and keyboard navigation.

### ‚ö° Technical Features
- Fetches weather data dynamically using **Open-Meteo API**.
- Dynamically updates:
  - Weather icons based on weather codes.
  - Forecast weekdays and temperatures.
  - Hourly forecast times and temperatures.
- **Local unit conversion** without extra API calls.
- Lightweight, fast, and optimized for small bundle sizes.
